# probemap — backlog / идеи

Короткий список на будущее (не блокирует релизы).

## Сделано (недавно)

- Палитра: вместо текста «нет на карте» — кнопка **Добавить**; стиль кнопки — нейтрально-серый.
- Добавление из палитры: раскладка по **левому краю видимой области** сверху вниз, затем колонка правее; запасной поиск спиралью.
- Клик по строке в палитре (узел на карте): **панорама к узлу** (`setCenter`), масштаб без изменений.
- Статусы по пробам, счётчик «Пробы», кружки blackbox, подсветка рамки узла, i18n — см. историю коммитов.
- **Подсказка под палитрой**: список узлов из мониторинга; ПКМ — область, узел из мониторинга или «Без метрик» (`white-space: pre-line`).
- **Терминология в UI**: «узел» в ключевых строках настроек/проекта/лейбл-карты (где имелось в виду карта, а не backend `service`).
- **Превью селектора в настройках** с учётом **фильтра активного проекта** (пары `project_filter_pairs` на `/api/config/preview-selector`, как на сервере для `/api/projects/.../services`).
- **`GET /health`** на бэкенде — ответ `{"status":"ok"}` для K8s и т.п.
- **Индикатор покрытия blackbox** в тултипе узла: строка `Blackbox: present/expected` (сколько источников дали метрики из ожидаемого списка).
